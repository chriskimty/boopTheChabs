# About
<p><a href="https://boopthechabs.netlify.app/"> Boop👆 the Chabs</a> is a reaction timer created with VueJS.</p> 

<p>Users have one job: to boop the 'Chabs' as fast as they can to become a "Legendary Booper🏆".</p>
